    Evaluating output data

    The Data Browser tool lists all input and output data. Its navigation is similar to the parameter editor’s Data category (see Importing data).


    Depending on the data type, the data browser offers actions such as importing the data back into ImageJ via Bioformats Importer. The “…” button opens a list of other available options.


    Following actions are available by default:


    Action

    Description

    Bioformats import

    Only available for OME TIFF data.

    Imports the OME TIFF via the Bioformats importer.

    Set current image name

    Only available for image data.

    Sets the name of the currently selected image in ImageJ to a name unique to the data.

    Open folder

    Opens the folder that contains the data.

    Copy path

    Copies the folder path into the clipboard.
